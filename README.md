**DISCORD BOT FOR UBC COOKING CLUB**

Created by me for the UBC Cooking Club Discord, if you have any issues find some way to message me or any director at UBC Cooking Club.

**COMMANDS**
- !roles: Returns list of all exec positions.
- !purge: Deletes (up to) 100 last messages.
- !setup: Dms author instructions for exec setup.
- !poll *topic*: Quick yes/no reaction poll.
- !rename (name): To change your nickname to (name).
- !leave (position): To remove role.
- !join (position): To begin exec onboarding process.

*note that most roles above can only be used by temp-execs and up, with some only availible to Admins and Directors.*
